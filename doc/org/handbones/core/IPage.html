preShutdown()method
public function preShutdown():void

Executes before shutdown. At default this will: Dispatch PageEvent.SHUTDOWN. Remove all children from this page. Unmap all injections mappings made on this page. Unmap all commands mappings made on this page. Unmap all mediators mappings made on this page. Unmap all views mappings made on this page.

shutdown()method 
public function shutdown():void

Executes when page is removed from the stage. At default this will dispatch PageEvent.SHUTDOWN_COMPLETE. Mostly overriding won't be needed as the preShutdown unmaps everything mapped on this page.

startup()method 
public function startup():void

Executes when page is added to the stage. At default this will dispatch PageEvent.STARTUP_COMPLETE. This is ment to be overritten for your custom mappings to be made. This will decouple this page from the rest of the site and prevent cross compiling.
